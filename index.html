    The purpose of this project was to analyze how weather changes the closer one gets to the equator. For this analysis I pulled data from OpenWeather API to retrieve a dataset from 500 cities. To plot the various data regarding latitude vs weather I used Matplotlib. This website provides source data and visualizations of the analysis.
